# Bayo.net - Dark Web Intelligence Simulator

[![Python](https://img.shields.io/badge/python-3.11-blue)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/flask-2.3-green)](https://flask.palletsprojects.com/)

Bayo.net is an **interactive web application simulating dark web and deep web searches**. Users can enter three words or phrases to see simulated results, check risk levels, and upload CVs or project files for sharing.

This project is ideal for **developers, cybersecurity enthusiasts, and anyone looking to showcase their portfolio interactively**.
